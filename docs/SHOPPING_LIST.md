## Shopping Checklist

### Before Ordering

- [ ] Decide quantity (1, 10, or 100)
- [ ] Check budget
- [ ] Verify supplier ratings
- [ ] Compare prices across stores
- [ ] Check shipping time
- [ ] Verify return policy

### When Ordering

- [ ] Order 10% extra components (for failures)
- [ ] Buy tools if first time
- [ ] Get variety of jumper wires
- [ ] Order extra batteries (for testing)
- [ ] Get SIM cards with data activated

### After Receiving

- [ ] Test each component individually
- [ ] Check for damaged items
- [ ] Verify specifications match
- [ ] Keep invoices for warranty
- [ ] Store components properly
